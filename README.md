## Data Structure

**Clients:**
- Name, email, company
- Notes
- Creation date

**Projects:**
- Name, description
- Linked client
- Budget
- Status (pending/active/completed)

**Invoices:**
- Invoice number
- Linked project
- Amount
- Status (pending/paid)
- Due date
